# cc-watch

tmux 上で動いている Claude Code セッションのうち、**ユーザー入力を待っているもの**をスキャンして一覧表示し、選択するとそのペインにジャンプするツールです。

複数の Claude Code セッションを並行して走らせているとき、どのセッションが確認待ちになっているかをすばやく把握できます。

## 検出する待機状態

| アイコン | ラベル | 検出パターン例 |
|---------|--------|---------------|
| `!` | 確認待ち | `Do you want to proceed`, `Allow`, `Deny` |
| `?` | Yes / No | `(y/n)`, `(yes/no)` |
| `↵` | Enter待ち | `Press Enter`, `[Enter]` |
| `$` | コスト確認 | `API Cost`, `tokens used` |
| `›` | 入力待ち | プロンプト (`❯`) |

## 必要なもの

- **tmux** — セッション管理
- **zsh** — 実行シェル
- **fzf** (任意) — プレビュー付きファジーセレクター。未インストールの場合は対話式メニューにフォールバック

## インストール

```sh
# リポジトリをクローン
git clone https://github.com/m1ng112/cc-watch.git

# パスの通った場所にリンク
ln -s "$(pwd)/cc-watch/cc-watch" ~/.local/bin/cc-watch
```

## 使い方

tmux セッション内で実行します。

```sh
cc-watch
```

- **fzf がある場合** — ファジーセレクターが起動し、右側にペインのプレビューが表示されます。選択すると該当ペインに切り替わります。
- **fzf がない場合** — 番号付きの対話式リストが表示されます。番号を入力して Enter で該当ペインにジャンプします。
- **待機中のセッションがない場合** — `✓ 待機なし` と表示されます。

### tmux キーバインド例

```sh
# prefix + w で cc-watch を起動
bind-key w run-shell -b 'cc-watch'
```
